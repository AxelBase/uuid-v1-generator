Extracting Creation Time from Any UUID v1

Published: November 28, 2025

One of the most powerful features of UUID v1 is that the creation time is permanently embedded in the identifier itself — no database lookup required.

This tool instantly decodes any valid v1 UUID and shows you:

  • Human-readable UTC timestamp (e.g., “2025-11-28 14:32:10.123 UTC”)
  • Unix milliseconds since 1970
  • 100-nanosecond precision from the original Gregorian epoch

How It Works

The first 60 bits of a v1 UUID form a counter of 100ns intervals since October 15, 1582. By extracting and reordering these bits, we reconstruct the exact moment the UUID was generated.

Real-World Uses

  • Debugging distributed systems
  • Correlating logs across services
  • Forensic analysis of data breaches
  • Verifying event ordering without timestamps
