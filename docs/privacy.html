Privacy Policy

Privacy Policy

Effective Date: November 28, 2025

Welcome to the UUID v1 (Timestamp) Generator by AxelBase. This tool was built with one core principle: your data never leaves your device.

1. Zero Data Collection

We do not collect, store, transmit, or process any personal information — period.

  • No IP addresses are logged
  • No browser fingerprinting
  • No cookies, localStorage persistence, or tracking pixels
  • No analytics (Google Analytics, Plausible, Umami, etc.)
  • No server-side logging of any kind

All UUID generation, timestamp decoding, bulk exports, and clipboard operations happen entirely in your browser using client-side JavaScript.

2. 100% Client-Side Operation

When you use this tool:

  • UUIDs are generated using crypto.getRandomValues() in your browser
  • Timestamps are extracted using pure JavaScript (no network calls)
  • Bulk downloads are created via Blob and URL.createObjectURL()
  • Copy-to-clipboard uses the native Clipboard API

Nothing is ever sent to any server — not even error reports.

3. Offline-First & PWA

This site is a Progressive Web App. After your first visit, it works completely offline. The service worker caches all assets locally — ensuring zero telemetry even when disconnected.

4. Third-Party Services

We use only one external resource: Bootstrap 5.3.3 via jsDelivr CDN (for styling). This CDN does not set cookies or track users when used in this static context. No other third-party scripts are loaded.

5. Open Source Transparency

The entire source code is publicly available on GitHub. You can verify for yourself that no tracking or data collection exists. We encourage security researchers and privacy advocates to audit the repository.

6. Children's Privacy

This tool is safe and appropriate for all ages. Since no data is collected, it fully complies with COPPA and similar regulations by default.

7. Changes to This Policy

If we ever change this policy (extremely unlikely), the new version will be posted here with an updated effective date.

8. Contact

Questions? Open an issue on our GitHub repository — we respond quickly.

Privacy isn’t a feature — it’s the foundation.
